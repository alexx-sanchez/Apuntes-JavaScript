Arrays

.push()

Agrega uno o más elementos al final de un array y devuelve la nueva longitud del array.

Código usado: frutas.push("naranja")

.pop()

Elimina el último elemento de un array y lo devuelve. Modifica el array.

Código usado: frutas.pop()

.shift()

Elimina el primer elemento de un array y lo devuelve. Modifica el array.

Código usado: frutas.shift()

.unshift()

Agrega uno o más elementos al inicio de un array y devuelve la nueva longitud.

Código usado: frutas.unshift("kiwi")

.sort()

Ordena los elementos de un array en su lugar y devuelve el mismo array.

Código usado: numeros.sort((a, b) => a - b)

.reverse()

Invierte el orden de los elementos del array. El primer elemento pasa a ser el último, y viceversa.

Código usado: numeros.reverse()

.every()

Verifica si todos los elementos del array cumplen con una condición. Devuelve true o false.

Código usado: edades.every(edad => edad >= 18)

.some()

Verifica si al menos uno de los elementos cumple una condición. Devuelve true o false.

Código usado: edades.some(edad => edad < 18)

.find()

Devuelve el primer elemento que cumpla una condición. Si no encuentra, devuelve undefined.

Código usado: numerosGrandes.find(num => num > 15)

.findIndex()

Devuelve el índice del primer elemento que cumpla una condición. Si no encuentra, devuelve -1.

Código usado: numerosGrandes.findIndex(num => num > 15)

.map()

Crea un nuevo array con los resultados de aplicar una función a cada elemento del original.

Código usado: [1, 2, 3].map(num => num * 2)

.reduce()

Reduce un array a un solo valor usando una función acumuladora.

Código usado: [1, 2, 3, 4].reduce((acum, actual) => acum + actual, 0)

.filter()

Devuelve un nuevo array con los elementos que cumplan una condición.

Código usado: [1, 2, 3, 4, 5, 6].filter(num => num % 2 === 0)

Clase Alumno

Descripción: La clase Alumno modela a un alumno con propiedades como nombre, apellidos, fecha de nacimiento, email, ciclo y notas. También tiene métodos para calcular la edad, la media de las notas y mostrar información del alumno.

                
class Alumno {
    constructor(nombre, apellidos, fechaNacimiento, email, ciclo, notas) {
        this.nombre = nombre;
        this.apellidos = apellidos;
        this.fechaNacimiento = new Date(fechaNacimiento);
        this.email = email;
        this.ciclo = ciclo;
        this.notas = notas;
    }

    calcularEdad() {
        const hoy = new Date();
        let edad = hoy.getFullYear() - this.fechaNacimiento.getFullYear();
        const mesDiferencia = hoy.getMonth() - this.fechaNacimiento.getMonth();
        if (mesDiferencia < 0 || (mesDiferencia === 0 && hoy.getDate() < this.fechaNacimiento.getDate())) {
            edad--;
        }
        return edad
    }

    calcularMedia() {
        const ponderaciones = this.ciclo === "DAM" ? [0.3, 0.2, 0.3, 0.2] : [0.2, 0.4, 0.2, 0.2];
        return this.notas.reduce((total, nota, i) => total + nota * ponderaciones[i], 0).toFixed(2);
    }

    toString() {
        return `${this.nombre} ${this.apellidos}`;
    }

    infoAlumno() {
        return `${this.nombre} ${this.apellidos} | Edad: ${this.calcularEdad()} | Ciclo: ${this.ciclo} | Media: ${this.calcularMedia()}`;
    }
}
